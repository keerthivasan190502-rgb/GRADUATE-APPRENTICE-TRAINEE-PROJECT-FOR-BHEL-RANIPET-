## \ud83d\udccb About This Report

This repository contains the complete **Apprenticeship Project Report** submitted as part of the one-year training programme at **BHEL's Stores and Material Management Department**. The report documents three real-world improvement projects undertaken using **SAP ERP \u2013 Materials Management (MM) Module**.

---

## \ud83d\uddc2\ufe0f Projects Overview

### \u2705 Project 1: SAP-Based Inventory Optimization
> **Objective:** Reduce excess inventory and prevent stock-outs using SAP MM data

- Applied **ABC & XYZ Analysis** on SAP stock data
- Used transactions: `MB52`, `MMBE`, `MM60`, `MB51`
- Identified **~15% slow-moving** and **~8% non-moving** materials
- Calculated optimal **Reorder Levels**, **Safety Stock**, and **EOQ**
- **Expected saving:** \u20b935\u201350 Lakhs annually

---

### \u2705 Project 2: Stores Layout Optimization
> **Objective:** Reduce material picking time by reorganizing stores using SAP movement data

- Extracted movement frequency using `MB51` (12-month analysis)
- Classified materials into **High / Medium / Low Frequency** zones
- Proposed **Zone-based layout** (Zone A to Zone G)
- Applied **5S Methodology** for workplace organization
- **Expected improvement:** 25\u201330% reduction in picking time

---

### \u2705 Project 3: SAP Material Tracking & Traceability System
> **Objective:** Document end-to-end material lifecycle from GRN to production

- Mapped full SAP transaction chain: `ME51N` \u2192 `ME21N` \u2192 `MIGO` \u2192 `MB1A` \u2192 `MI01`
- Created **8 Standard Operating Procedures (SOPs)** for stores operations
- Identified **4 key traceability gaps** with recommendations
- Demonstrated complete **audit trail** using `MB51`, `MB52`, `ME2M`, `MMBE`
